Instrucciones para descargar el codigo fuente desde GitHub: git clone https://github.com/jhmesa/psp2.1-program-7.git
Instrucciones para probar el programa en heroku: Ingrese en el navegador la url https://program7.herokuapp.com/calculation?fileName=<valor>&estimatedProxy=<valor>
												Ejemplo: https://program7.herokuapp.com/calculation?fileName=test_1.txt&estimatedProxy=386
Instrucciones para compilar el codigo fuente: Ejecute el comando mvn package desde psp2.1-program-7
Instrucciones para desplegar la aplicación localmente:  Ejecute el comando heroku local web desde psp2.1-program-7
Instrucciones para ejecutar el programa: Ingrese en el navegador la url localhost:5000/calculation?fileName=<valor>&estimatedProxy=<valor>
										 Ejemplo: localhost:5000/calculation?fileName=test_1.txt&estimatedProxy=386
